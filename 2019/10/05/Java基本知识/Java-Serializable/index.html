【Java】Serializable

发表于 2019-10-05 更新于 2019-10-10 分类于 Java

Java序列化，将Java对象转换为字节数组，便于存储传输
反序列化，将字节数组转换为Java对象

  1. 如果一个类能够序列化，他的子类能够序列化
  2. 如果子类实现了Serializable接口，父类没有实现，父类不能被序列化，子类可以
  3. 由static修饰的类的成员，transient修饰的对象代表对象的临时数据，不能被序列化
